# Averager

**Averager** is a simple way to calculate averages of values.

```py
>>> average([1, 2, 3])
2

>>> weighted_average([(1, 2), (2, 3)])
1.6
```

## Installation

Averager is available on PyPI:

```console
$ python -m pip install averager
```

Averager officially supports Python 2.7 & 3.5+.
